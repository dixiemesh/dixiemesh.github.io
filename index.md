# Welcome to Dixie Mesh

Dixie Mesh is a community-run [LoRa](https://en.wikipedia.org/wiki/LoRa) mesh
network built on [MeshCore](https://meshcore.io). It lets people send
messages across town — and beyond — without relying on cellular service, the
internet, or any central infrastructure. Just low-power radios passing messages
node to node.

Whether you're a seasoned radio operator, an off-grid tinkerer, or simply
curious about resilient communication, you're in the right place.

## What is a mesh network?

In a mesh network, every node can relay messages for every other node. There's
no single point of failure and no monthly bill — the network is the people who
run it. The more nodes that join, the further messages can travel and the more
reliable the whole system becomes.

## Why Dixie Mesh?

- **Independent** — works when the internet and cell towers don't.
- **Low cost** — built on inexpensive, low-power LoRa hardware.
- **Community owned** — no company, no subscription, no gatekeepers.
- **Open** — built on the open MeshCore project.

## Get involved

Dixie Mesh grows with every new node. If you'd like to join the network, set up
your own device, or just learn more, follow along as we add guides here.
